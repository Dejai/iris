IRIS Beta Tasks

Below are a set of beta tasks for features to be released in 15.13


Please complete these tests by: November 13th, 2015

Advanced Filters


Using the Advanced Filter button, complete the following tasks

  1. Locate Offers that are are Paid creatives
  2. Find all paid creatives that were created between January 1st 2015 and January 31st 2015.
  3. Show those creatives that were last used between July 1st 2015 and September 1st 2015.
  4. View those pages that are for a product, and are either active OR shelved.
  5. Find all the creatives that are for Lead Gen Items and are Inactive.
  6. Show only those paid emails that have an active status.
  7. Look up the paid banner ads that are for the same item (choose any item code).
  8. Now look up paid banner ads for multiple items (choose more than one item code); Sort by Item code.
  9. Try to find any lead gen creative that meets ALL of the following criteria:
    • Created between January 1st and Today's date
    • Last used between August 15th and yesterday's date.
    • Active, Shelved or Inactive.
    • Sort them by last used date in ascending order (most recent date last).

Multiple Status Change


  1. Try to change multiple status of your creatives without filtering.
  2. Try again to filter your creatives by only filtering the Creative Type; Now filter by Creative Type and Item and try again.
  3. Filter your emails to the same Item, Creative Type and Status (choose any status). Change the status of 3 of these to another status.
  4. Filter your pages to the same Item, Creative Type, status and OFFER. Then change 3 of these to another status.
  5. Filter your offers to the same Item, Creative Type, Status and Function.
  6. Add three creatives for the same Item
    • Set them all to be shelved at different times in the future (2 month, 4 months and 6 months respectively).
    • Now, try to change the status of just these three creatives. Set them to expire 5 months from now. Hint
