IRIS Beta Tasks & Scenarios

Purpose

This webpage presents scenarios and tasks that highlight new permissions and an update to the effort workflow in
IRIS 4.0 Phase 1 Release 3

It is intended to provide users with an opportunity to familiarize themselves with the variations of permissions and the new interface.

Furthermore, this document serves as an opportunity for testers to provide feedback on these changes.



The deadline to submit feedback is Friday, January 22nd 2016



How To Use This Webpage

Get Started With Tasks

How To Use This Webpage

Comments/Notes Section:

At the end of each scenario/task, you will find a comments section. We encourage you to use this section to record any comments, suggestions, questions or concerns you have while testing the associated scenario/task.

If you discover any issues with the system, or if you have any comments/suggestions, please submit a ticket to Zendesk via support@pubsvs.com.

Note: Please include “#IRISBETA” or “#irisbeta” anywhere in your email subject line or body.

Remember - the deadline to submit feedback is Friday, January 22nd 2016

Scenarios:

For tasks, you should ensure that you have all permissions. For scenarios, we recommend that you modify your permissions accordingly to compare and contrast your abilities in the system.

Only Admin accounts can freely change their permissions. If you have a User account, please partner with someone who has an Admin account to change your permissions.

Key Terminology:

Default The Creative that will be automatically associated with a selected Creative. Can be a default order form (for pages) or a default page (for emails and ads)
Order Form Synonymous to an Offer in IRIS
Page 1 This is the first page a customer sees after clicking the promo link. It can be the landing page or the order form
Page 2 The second page a customer sees after your main page. It could be a redirect page or the order form.
Page 3 This refers to the Order Form used. This field shows up if Page 2 is a redirect page.
Get Started With Tasks

New Effort Workflow


Task #1:     Space Ad Efforts

Description:     Setting up a Space Ad effort using the new Page fields. The Ad is selected first, using the Advertisement pop-up window.

Steps:

  1. Select the ‘Effort’ tab.
    1. This is permission based. You must have “Create & Edit Efforts” permission.
  2. Enter/Select a date.
  3. Select Internal List or Website as Effort Destination.
  4. Select Item Type (Subscription or Product).
  5. Select Internal Item Code.
  6. Select Space Ad as the Effort Type
  7. Select a List
  8. Select a Space Ad using the Select One button.
  9. Select a Page in the Page 1 Field.
    1. If defaults have been set, Page 2 & Page 3 will automatically be filled with the redirect page and order form respectively.
    2. If Page 1 does not have a redirect page, Page 2 will be the order form, provided there is a default.
  10. Create the Effort.

New Effort Workflow


Task #2:        Order Form Only Effort

Description:        Setting up an effort using the new Page fields. This effort uses an order form in the page 1 field to make it an Order Form Only effort.

Steps:

  1. Select ‘Effort’ tab
    1. This is permission based. You must have “Create & Edit Efforts” permission.
  2. Enter/Select a date
  3. Select Internal List or Website as Effort Destination.
  4. Select Item Type (Subscription or Product).
  5. Select Internal Item Code
  6. Select Ed Mention as the Effort Type.
  7. Select a List
  8. Select an Order Form in Page 1 Field

    1. NOTE: You no longer have to click the ‘Order Form Only’ checkbox – it no longer exists. By selecting an Order Form in the page 1 field, the effort becomes an order form only (as noted by the Row Color field), and it will be labeled as such on the schedules page;

  9. Complete the remaining fields with any values of your choice.
  10. Create the Effort.

Task #3:        Change Defaults on Efforts

Description:        Setting up an effort using the new Page fields. This task focuses on changing defaults that were set for a creative. This is a permissioned based ability.

Steps:

  1. Select ‘Effort’ tab
    1. This is permission based. You must have “Create & Edit Efforts” permission.
  2. Enter/Select a date
  3. Select Internal List or Website as Effort Destination.
  4. Select Item Type (Subscription or Product).
  5. Select Internal Item Code
  6. Select Banner Ad as Effort Type
  7. Select List
  8. Select Banner Size
  9. Select a Banner Ad using the Select One button
  10. Select a Page in Page 1 Field
    1. If defaults have been set, Page 2 & Page 3 will automatically be filled with the redirect page and order form respectively.
    2. If Page 1 does not have a redirect page, Page 2 will be the order form, provided there is a default.
  11. Change the default Order Form
  12. Complete the remaining fields with any values of your choice.
  13. Create the Effort.

Permissions

As an IRIS user, permissions control your limitations and flexibility throughout the system
from setting up entities (offers, pages, etc.) to creating efforts and scheduling mailings.


The following diagrams highlights the relationship between permissions associated with your experience on the creative side and the effort side.

Creative Permissions

Effort Permissions



These two new permissions overlap with existing permissions to provide a customizable experience in the system.


To help you become accustomed to these new permissions, and their relationship with existing ones, we have provided scenarios!

Begin Junior Level Marketer Scenarios

Junior Level Marketer


Scenario #1:   Access to Ad with a Default

Description:   This scenario focuses on a Junior Level Marketer being able to create an Effort using a Space Ad that has a default Page.

Roles:

Senior Level Marketer (SM)

Junior Level Marketer (JM)

  1. JM: Verify that you cannot add, edit, copy or grant access to any creative.
    1. You should not be able to.
  2. SM: Grant the JM access to an Internal Space Ad that has a default page.
    1. Inform the JM of the Item and Name for the Space Ad you gave them access to.
  3. JM: Create a Space Ad Effort
    1. Use Internal List or Website as the Effort Destination
    2. Select the given Item Code
    3. Select Space Ad as Effort Type
    4. Select a Space Ad using the ‘Select One’ button.
      1. You should be able to select the Ad you were given access to.
  4. SM: Remove the user’s access to that Space Ad.
  5. JM: Refresh your page and try to select that Ad again in a Space Ad effort.
    1. You should no longer be able to.
