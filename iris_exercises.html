IRIS Practice Exercises

The following excercises are intended as practice to familiarize yourself with the IRIS user interface and workflow.


Exercise One

PRE-CONDITIONS: Go to the publishing services instance of IRIS (iris.pubsvs.com) and create a new offer for the item LD1.

Tasks:

  1. Create an offer in IRIS with 2 main prices / choices.

  2. Attach a cross sell offer to the offer in step 1. It can be any item other than LD1

  3. Create a piece of HTML content using the offer from step 1

  4. Create a piece of video content and attach the HTML version to that content

  5. No create a dedicated effort going to MC TEST LIST with the target IRIS_TEST_LIST-IRISUSER_1 using the version you just created. Complete the following:
    1. Enter an effort tag titled '[enter your name]'s First Test (trick question)
    2. Create and save a lift note from the effort level
    3. Enter a note on the effort level
    4. Create and assign an effort row color
    5. Create and assign the effort to a testing group
    6. Give yourself receive domain request email and paid code email permissions.

  6. Go to My Queue and request the promo code for the dedicated effort that you just created. Click on the link in the promo code request email and enter a code (it can be random)

  7. Go back to the effort, copy it and assign it to a multivariate group but do not add a split

  8. Do not add additional multivariate efforts

  9. Repeat step 7 with the new effort you created

  10. Now go to the mailing just created (you may not have known it but you just created a mailing)

  11. Send an IRIS test email to yourself!
NOTE: Your comments will not be saved if you exit/refresh the page, so be sure to copy & paste them elsewhere when you're done with the exercise.

Exercise Two

NOTE: Exercise 2 involves lead gen training. It will be more difficult than the first one because the lead gen application leaves little to be desired (and we know that lead gen in 4.0 will be completely different).


PRE-CONDITION: Go to the publishing services instance of IRIS (iris.pubsvs.com)

Tasks:

  1. This offer should include a cross sell for the list IRIS.

  2. Create a lead gen offer using the IRIS Test List and attach the co-reg from step 1.

  3. Create a signup page that also includes a checkbox for the coreg (I don't believe the system supports this through the WYSIWYG, you may need to code this yourself). I can find examples if you need them.

  4. Create a signup page that does not have a co-reg checkbox.

  5. Create a confirmation page that includes a co-reg signup form.

  6. Create a co-reg confirmation email

  7. Create a confirmation email

  8. Create a report (this may be out of sequence but the system should prompt you to create all of these entities).

  9. Now go to the effort level and create an effort using the signup page from step 3 using the confirmation page from step 5.

  10. Copy the effort created in step 9 and create a new effort but this time use the page from step 4. Select the co-reg confirmation page you created in step 5 and add this effort to the multivariate group.

  11. Go to my queue and request a promo code for those two efforts.
NOTE: Your comments will not be saved if you exit/refresh the page, so be sure to copy & paste them elsewhere when you're done with the exercise.
